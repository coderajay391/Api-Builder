A small utility project to help you design, test, and document APIs faster.

## Features

- Create and organize API endpoints
- Quick request/response testing
- Basic documentation in README form

## Getting Started

1. Install dependencies (if applicable):
   - `npm install`
2. Run the app:
   - `npm start`
   - or `npm run dev`
3. Open the API builder UI / run the server and start creating endpoints.

## Usage

1. Define an endpoint (method, path, and payload).
2. Configure any required headers/auth.
3. Send a request and inspect the response.
4. Save/export the endpoint details.

## Project Structure

- `src/` - application code
- `routes/` or `api/` - API endpoint definitions
- `README.md` - documentation
